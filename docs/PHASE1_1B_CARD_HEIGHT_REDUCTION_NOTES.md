# Phase 1.1B — Reduced card heights

This patch reduces the vertical footprint of KPI and chart cards so card height tracks the amount of data more closely.

## Changes
- Reduced KPI card minimum height
- Reduced shared card padding
- Tightened KPI internal spacing
- Reduced chart card minimum heights
- Reduced chart plot wrapper minimum heights
- Slightly tightened gaps and corner radius for a denser executive layout
