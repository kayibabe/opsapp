SIDEBAR SUBMENU INDENT PATCH

Purpose
- Moves all sidebar sub-menus slightly inward to create clearer hierarchy under their parent hub.

Files included
- app/static/assets/css/sidebar-submenu-indent-override.css

How to apply
- Load this stylesheet after base.css, or copy the rules into the end of app/static/assets/css/base.css.

Effect
- Child menu groups are nudged inward
- Individual child links get a little more left padding
- Dot markers / before-elements move with the new indentation
