TOPBAR DATETIME RIGHT-SPACING NOTES

Purpose
- Adds breathing space on the right side of the live date/time block so it no longer touches the page edge.

Files included
- app/static/assets/css/topbar-datetime-spacing-override.css

How to apply
Option 1
- Link this stylesheet after your existing base.css in app/static/index.html

Option 2
- Copy the CSS rules into the end of app/static/assets/css/base.css

Effect
- Adds right padding to the top-bar right action zone
- Adds a small right margin to the live date/time display
- Keeps the alert bell and other controls visually aligned
