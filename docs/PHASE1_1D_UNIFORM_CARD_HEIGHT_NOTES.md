# Phase 1.1d Uniform Card Height Patch

This patch standardizes the main dashboard and report card families to the same compact shell height as the approved KPI reference card.

## What changed
- Sets a shared compact minimum height for KPI and non-chart cards
- Aligns border radius and padding across card families
- Keeps chart cards taller only where needed for graph readability
- Tightens shared heading and legend spacing so the whole system feels more consistent

## Files
- `app/static/assets/css/base.css`
- `PHASE1_1D_UNIFORM_CARD_HEIGHT_NOTES.md`

## Apply
Replace the matching CSS file, then hard refresh the browser.
