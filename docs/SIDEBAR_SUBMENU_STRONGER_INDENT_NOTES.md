SIDEBAR SUBMENU STRONGER INDENT PATCH

Purpose
- Moves submenu links further inward on the right-hand side so they are visually distinct from main menu items.

Files included
- app/static/assets/css/sidebar-submenu-stronger-indent-override.css

How to apply
- Load this stylesheet after base.css and after any earlier sidebar indent override.
- Or copy these rules into the end of app/static/assets/css/base.css.

Effect
- Child menu groups shift slightly right
- Child links gain stronger left padding
- Dot markers move with the new indentation
